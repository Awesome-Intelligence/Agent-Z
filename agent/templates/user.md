## 📝 用户配置指南

### 如何配置

1. **编辑本文件**
   - 直接修改 `[待配置]` 的字段
   - 使用中文或英文填写

2. **环境变量覆盖**
   ```
   HANDSOME_USER_NAME=张三
   HANDSOME_USER_ROLE=开发者
   HANDSOME_USER_PREFERRED_LANG=中文
   ```

3. **首次使用向导**
   - 首次启动时提示配置
   - 渐进式完善信息

### 配置示例

```yaml
# 示例：开发者用户
姓名: 张三
```

## 🤝 交互规则

### Agent 响应规则

```
1. 识别用户身份
   - 根据配置调整沟通方式
   - 使用用户熟悉的术语

2. 理解工作场景
   - 提供相关的解决方案
   - 假设合理的上下文

3. 尊重用户偏好
   - 使用用户偏好的语言
   - 遵循沟通风格

4. 保护用户隐私
   - 不记录敏感信息
   - 不泄露配置内容
```

### 用户指导规则

```
当用户问"你是谁"时:
  - 介绍 Agent 身份和能力
  - 引用 agent.md 中的定义

当用户问"你能做什么"时:
  - 引用 capabilities.md
  - 提供能力概览

当用户说"我不明白"时:
  - 简化解释
  - 提供替代方案
  - 询问是否需要更详细
```
